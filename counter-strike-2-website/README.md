# Counter Strike 2 Website

Este proyecto es una página web dedicada a Counter Strike 2, un popular juego de disparos en primera persona. La web incluye información sobre el juego, sus armas, mapas y más.

## Estructura del Proyecto

El proyecto tiene la siguiente estructura de archivos:

```
counter-strike-2-website
├── src
│   ├── index.html          # Página principal del sitio web
│   ├── pages
│   │   ├── home.html      # Página de inicio con información general
│   │   ├── weapons.html   # Página que muestra las armas disponibles
│   │   └── maps.html      # Página con información sobre los mapas
│   ├── components
│   │   ├── header.html     # Encabezado del sitio web
│   │   └── footer.html     # Pie de página del sitio web
│   ├── assets
│   │   ├── css
│   │   │   └── styles.css  # Estilos CSS del sitio web
│   │   ├── js
│   │   │   └── main.js     # Código JavaScript para interacciones
│   │   └── fonts           # Fuentes personalizadas
│   └── data
│       └── weapons.json    # Datos sobre las armas en formato JSON
├── package.json            # Configuración de npm
├── .gitignore              # Archivos y directorios a ignorar por Git
└── README.md               # Documentación del proyecto
```

## Instalación

1. Clona el repositorio en tu máquina local.
2. Navega al directorio del proyecto.
3. Ejecuta `npm install` para instalar las dependencias necesarias.

## Uso

Para iniciar el proyecto, abre `src/index.html` en tu navegador. Navega a través de las diferentes páginas para explorar la información sobre Counter Strike 2.
